# 宇航工具箱 - Chrome浏览器插件

宇航工具箱是一个功能强大的Chrome浏览器插件，集成了多种实用工具，帮助用户提高网页浏览和文本处理效率。

## 主要功能

1. **转JSON功能**：将任意文本智能转换为结构化的JSON格式
2. **翻译功能**：支持中、英、日、韩、法五种语言之间的互译
3. **文本总结**：自动分析并总结选中的文本或整个网页内容
4. **MD转Word**：将Markdown文件转换为Word文档格式，支持表格、代码块等元素

## 技术特点

- 所有功能基于DeepSeek大模型实现，确保高质量的文本处理效果
- 简洁美观的太空主题UI设计
- 支持Chrome侧边栏展示，不干扰浏览体验
- 响应速度快，支持错误处理和状态反馈

## 使用方法

### 安装插件

1. 下载插件文件
2. 打开Chrome浏览器，进入扩展程序页面 (chrome://extensions/)
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"，选择插件文件夹

### 配置API密钥

1. 安装插件后，点击插件图标打开侧边栏
2. 点击右上角设置图标
3. 输入您的DeepSeek API密钥（需要先在DeepSeek官网申请）
4. 点击"保存"按钮

### 使用功能

#### 转JSON功能

1. 在插件面板中选择"转JSON"标签
2. 输入要转换的文本
3. 点击"转换为JSON"按钮
4. 查看结果并可选择复制

#### 翻译功能

1. 在网页上选中要翻译的文本
2. 右键点击，选择"宇航工具箱: 翻译选中文本"，或直接在插件面板中选择"翻译"标签
3. 选择目标语言
4. 点击"开始翻译"按钮
5. 查看翻译结果并可选择复制

#### 总结功能

1. 在网页上选中要总结的文本（若无选中则总结整个页面）
2. 右键点击，选择"宇航工具箱: 总结选中文本"，或直接在插件面板中选择"总结"标签
3. 点击"开始总结"按钮
4. 查看总结结果并可选择复制

#### MD转Word功能

1. 在插件面板中选择"MD转Word"标签
2. 上传Markdown文件或直接在文本框中输入Markdown内容
3. 选择转换选项（包含图片、保留代码块格式、转换表格、生成目录）
4. 点击"开始转换"按钮
5. 转换完成后点击"下载Word文档"按钮保存文件

## 插件结构

- `manifest.json` - 插件配置文件
- `background.js` - 后台服务脚本
- `content.js` - 内容脚本
- `api-service.js` - DeepSeek API服务
- `sidepanel.html` - 侧边栏界面
- `sidepanel.js` - 侧边栏交互逻辑
- `styles.css` - 样式表
- `images/` - 图标和图片资源
- `lib/` - 第三方库文件

## 注意事项

- 插件需要DeepSeek API密钥才能正常使用所有功能
- 处理大量文本时可能需要较长响应时间
- 确保网络连接稳定以获得最佳体验
- MD转Word功能在本地完成，不需要网络连接
- 代理功能需要相应的网络权限，请谨慎配置代理地址

## 新增功能

### 🌐 代理功能

插件现在支持网络代理和URL拦截功能：

**功能特性：**
- URL拦截规则配置
- 代理服务器设置  
- 请求重定向规则管理
- 支持正则表达式匹配

**使用方法：**
1. 点击"代理"标签页
2. 配置拦截URL模式（如：`/gateway/`）
3. 可选：设置代理IP地址并启用代理
4. 添加重定向规则：
   - 匹配前：要替换的字符串
   - 匹配后：替换成的字符串
   - 勾选"应用"启用该规则
5. 点击"保存"应用配置

**注意事项：**
- 代理功能需要谨慎使用，确保配置正确
- 建议先在开发环境测试规则
- 代理地址格式：IP:端口（默认端口80）

## 隐私声明

- 本插件不会收集或存储用户的个人信息
- API密钥仅存储在本地浏览器中
- 文本内容仅在处理时发送至DeepSeek API，不会用于其他用途
- MD转Word功能完全在本地处理，不会上传文件内容
